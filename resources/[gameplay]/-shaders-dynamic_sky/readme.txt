Description:
This resource adds a dynamic sky for your MTA.
The sun and moon positions depend on the ingame time.
Current (based on server time) moon phase is counted
on resource start.
The night sky created with spacescape-0.3. The moon phase
textures got from http://www.kalendarz-365.pl/ksiezyca/kalendarz-ksiezycowy.html
